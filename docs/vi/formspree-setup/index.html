Hướng dẫn cấu hình Formspree cho checklist AI Coding

Trang này giải thích cách kết nối form checklist với Formspree, đồng thời giữ site GitHub Pages an toàn khi chưa cấu hình endpoint thật.

Các bước cấu hình

  1. Tạo hoặc đăng nhập tài khoản Formspree.
  2. Tạo form mới cho AI Coding Workflow Checklist.
  3. Copy URL endpoint của Formspree.
  4. Mở config/email_capture.json.
  5. Đặt provider là formspree.
  6. Dán endpoint vào form_endpoint.
  7. Đổi enabled thành true.
  8. Chạy python main.py và python scripts/sync_site_output_to_docs.py.
  9. Commit và push static files đã cập nhật.
  10. Test gửi email thật từ trang checklist live.

Quy tắc an toàn hiện tại

Nếu enabled=false hoặc form_endpoint trống, form vẫn ở chế độ thiết lập và không gửi dữ liệu đi đâu.

FAQ

Có cần API key trong HTML không?

Không. Với Formspree, site static chỉ cần form endpoint công khai. Không đưa API key hoặc token riêng vào HTML.

Khi nào form bắt đầu gửi email thật?

Chỉ khi config/email_capture.json có enabled=true và form_endpoint là endpoint Formspree thật.

Nếu endpoint trống thì sao?

Form vẫn ở chế độ thiết lập, không POST đi đâu và không giả vờ lưu subscriber.
