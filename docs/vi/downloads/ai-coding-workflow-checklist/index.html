Checklist workflow AI coding

Dùng checklist này trước khi publish trang hoặc cập nhật dự án có AI hỗ trợ.

  1. Dùng Windsurf để dựng khung dự án nhanh.
  2. Dùng Codex để sửa/refine logic, kiến trúc và lỗi build.
  3. Kiểm tra language integrity cho English và /vi/.
  4. Kiểm tra SEO title, meta description, H1, canonical và hreflang.
  5. Build sitemap và đảm bảo không có /go/ trong sitemap.
  6. Sync output sang docs cho GitHub Pages.
  7. Submit sitemap trong Google Search Console.
  8. Tạo social drafts để duyệt thủ công.
  9. Theo dõi kết quả bằng UTM/tracking map.
  10. Cải thiện content quality dựa trên lỗi và dữ liệu.

FAQ

Checklist này có miễn phí không?

Có. Đây là tài liệu miễn phí để bạn tự kiểm tra workflow AI coding trước khi publish.

Form có gửi dữ liệu vào hệ thống tự động không?

Chưa. Website đang chạy static trên GitHub Pages nên hiện dùng mailto/fallback và chuẩn bị sẵn file subscribers.csv cho tích hợp sau.

Checklist có thay thế review thủ công không?

Không. Checklist giúp bạn nhớ các bước cần kiểm tra, nhưng vẫn cần tự xem lại code, SEO, sitemap và nội dung.
