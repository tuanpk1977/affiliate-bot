About MS Smile AI Review Hub

This site exists to review AI coding tools through real building work, not polished product demos.

Why this site exists

I am using Windsurf, Codex, Cursor, and GitHub Copilot while building this very review site. The goal is to document what actually helps: scaffolding, debugging, refactoring, SEO checks, language integrity, GitHub Pages deployment, and the cleanup work that demos rarely show.

The reviews are written from a builder workflow perspective. A tool can be fast and still create cleanup debt. Another tool can feel slower but be better when the repository is already messy. That tradeoff is what this site tracks.

What I track

  • Which tools speed up real project work.
  • Where AI coding tools break or repeat mistakes.
  • How SEO, hreflang, sitemap, and deployment checks behave in practice.
  • How social drafts and tracking can support affiliate research without spam.
