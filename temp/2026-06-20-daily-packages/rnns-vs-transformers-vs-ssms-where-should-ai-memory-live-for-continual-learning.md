## Why this matters now

## The memory question for AI

Continual learning systems need to decide where memory lives: inside a model, in a separate archive, or in a streaming buffer.

## RNNs and sequential state

RNNs maintain hidden state across time steps, but they are often limited by vanishing gradients and sequence length.

## Transformers and context windows

Transformers rely on attention over a fixed context window, which makes long-term memory expensive unless specialized memory architectures are added.

## SSMs and long-range recurrence

SSMs can deliver longer-range temporal dependencies with more efficient memory mechanisms, but they require careful engineering and are still maturing.

## Practical takeaway

For real-world continual learning, the best choice depends on whether you need immediate response speed, model-size efficiency, or scalable long-term memory.
